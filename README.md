
### 项目技术说明
本项目中，使用的是先后端分离。后端采用的是Spring boot 前端采用的是vue+element

环境说明：

jdk版本： 1.8及以上版本

node版本： 12.18.3及以上版本

数据库：mysql数据库5.7及以上版本

redis: 5.0.7及以上版本
